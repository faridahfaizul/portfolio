Project Overview

Do-It-Yourself (DIY) furniture based on augmented reality approach (DIYfAR) is a Final-Year Project proposed to help user to build a furniture with the use of AR platform.

The AR application will focus to use the markerless system technique, which is suitable in augmented reality that involves the use of GPS (Global Positioning System) and image recognition. Markerless system can be functioning at any places because they do not need any special labeling or extra reference such as marker-based system and it can be augmented on any real-time object. User can design the furniture that they intended to build, view it in real-time and receive information in order to build the furniture.

Project Limitations

  • This mobile application is only available for Android Operating System and not available for iOS and other mobile operating system.
  • This DIYfAR application are limited to small desks and tables for the 3D models. The measurements of the designs are different with each other.
  • This program restricts the user to use the readily available designs that are limited to just three colours for the customization features.
  • DIYfAR mobile application only has one language selection which is English.
  • The tutorial video is limited to one furniture only which is the Simple Round Table design due to the lack of time.

Research Methodologies

RAD model is reasonable for the activities in which the prerequisites are steady and straightforward. RAD underscores a short advancement cycle and is a straight successive programming improvement process utilizing a segment-based development approach. This DIYfAR project covers all of the phases that are listed in RAD model.

Conclusions

  • DIYfAR is proposed to help users to design and build furniture.
  • Augmented reality is used to help users to visualize the output / end product before continuing the project.

Recommendations for Future Work

  • Add a section in the table details page that allows user to post the completed table made by them and allows other users to view the photos.
  • Provide more picture from different view of the 3D model at the Overview page.
  • Allow users to add or create their own design in the application.
  • Give an option for the users to communicate. For example, chat features or comments section at the item so that user can ask for any questions and can also be answered by other users.
  • Add more categories of furniture and allow users to view more than one model in the AR mode.
  • Provide a more stable design to view in AR mode .
