Project Overview

Artliys is the design of mobile application that I did during completing the Google UX Design Certificate. It's main highlight is the audio feature for users to use when they are inside the gallery.

Artliys is a mobile application that provides audio feature when users are inside the galleries. This features can help galleries to share more information to the users and help users with reading disabilities. The application also allows users to control the audio playback. The goal of this project is to design a mobile app that can help users to get more information easily on artworks that is available in the art gallery with the help of audio feature.
