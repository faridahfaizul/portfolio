Project Overview

I started to have my own portfolio when completing the Coursera Course as it is one of the assignement. I explored some website builder and going through the templates helps me to find out how a portfolio should look like and how to display our works better.

I ended up using Wix for the portfolio assignment but changed to Google Sites after, as I find it easier to use. Since I love the template design that I use, I decided to recreate the template using my own skills which produced this website. I use Netlify to host this website for free using a custom name.
